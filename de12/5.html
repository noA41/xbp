第五回

  1. print("コンピューターの数字を当てろ！ドキドキ★じゃんけんバトル！！")
  2. import random
  4. ai = random.randint(1, 100)
  6. number = input("100までの数字を教えて下さい")
  8. if number == ai:
  9.     print("win!")
  10. else:
  11.     print("lose")
  13. print("正解は", ai, "でした。")
今回作成したコードは、ランダムな指定された数字を当てるゲームといういたってシンプルなゲームである。
02,04行目でaiの変数が1~100の間のランダムになるようにし、06行目でプレイヤー側に1~100の間の数字の入力を求めるというものである。
08~11行目ではif文をを用いてaiと入力の数字が同じならwinのメッセージが、違う場合はloseのメッセージが表示される。
(因みに数字があっていても全角入力の場合はloseと表示されるので半角で入力しなければwinのメッセージは出ない。)
